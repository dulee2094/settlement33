# 🚀 빠른 테스트 시작 가이드

## 현재 상태
- ✅ 서버: http://localhost:3000
- ✅ 가해자: offender@test.com / test1234
- ✅ 피해자: victim@test.com / test1234

---

## 1️⃣ 사건 생성 및 연결 (3분)

### Chrome (가해자)
```
1. 로그인: offender@test.com / test1234
2. "새 사건 생성" 클릭
3. 정보 입력 후 초대 코드 복사
```

### Edge (피해자)
```
1. 로그인: victim@test.com / test1234
2. "사건 참여" 클릭
3. 초대 코드 입력
```

---

## 2️⃣ 블라인드 제안 테스트 (5분)

### 양쪽 브라우저
```
1. 사건 상세 → "합의금 제안" 클릭
2. "합의금 제안하러 가기" 클릭
3. 금액 입력:
   - 가해자: 500만원
   - 피해자: 700만원
4. "제안 제출" 클릭
5. 분석 결과 확인
```

### 중간 합의 테스트
```
다음 라운드:
   - 가해자: 550만원
   - 피해자: 600만원
   → 차이 8.3% → 중간값 575만원 제안!
```

---

## 3️⃣ 실시간 채팅 (2분)

### Chrome (가해자)
```
1. "안심 채팅" 클릭
2. 메시지 입력: "안녕하세요"
3. 전송
```

### Edge (피해자)
```
1. "안심 채팅" 클릭
2. 메시지 실시간 수신 확인
3. 답장 전송
```

---

## 4️⃣ 사과문 (5분)

### Chrome (가해자)
```
1. "사과문" 클릭
2. "사과문 작성하러 가기" 클릭
3. 내용 작성
4. "디자인 미리보기 및 전송" 클릭
```

### Edge (피해자)
```
1. "사과문" 클릭
2. 도착한 사과문 확인
```
