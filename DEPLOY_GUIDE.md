# 웹사이트 실제 배포 가이드 (Deployment Guide)

작성하신 SafeSettlement 웹사이트를 전 세계 누구나 접속할 수 있도록 인터넷에 게시하는 방법입니다. 
서버 비용 없이 무료로, 가장 빠르고 쉽게 배포할 수 있는 **Netlify**를 추천합니다.

## 방법 1. 'Netlify Drop' 사용 (가장 쉬움, 1분 소요)

코드나 git 명령어를 몰라도 파일을 드래그 한 번으로 배포할 수 있습니다.

1.  **파일 준비**:
    *   작업하신 폴더(`합의 홈페이지`) 전체를 확인합니다.
    *   `index.html` 파일이 폴더 최상단에 있는지 확인하세요. (필수)

2.  **배포하기**:
    *   브라우저에서 **[Netlify Drop](https://app.netlify.com/drop)** 사이트에 접속합니다.
    *   로그인 (GitHub, Google 계정 등 사용).
    *   `합의 홈페이지` 폴더를 통째로 브라우저 화면의 **"Drag and drop your site folder here"** 영역에 끌어다 놓습니다.

3.  **완료**:
    *   잠시 후 배포가 완료되면 `https://random-name-123.netlify.app` 같은 인터넷 주소가 생성됩니다.
    *   이 주소를 클릭하면 스마트폰이나 다른 PC에서도 접속 가능합니다.

---

## 방법 2. GitHub 연동 (추천, 업데이트 관리 용이)

소스를 수정할 때마다 자동으로 사이트가 업데이트되길 원하신다면 이 방법을 사용하세요.

1.  **GitHub 저장소 업로드**:
    *   GitHub에 새 리포지토리(Repository)를 만듭니다.
    *   현재 폴더의 파일들을 해당 리포지토리에 업로드(Push) 합니다.

2.  **Netlify/Vercel 연동**:
    *   [Vercel](https://vercel.com) 또는 Netlify에 접속하여 가입합니다.
    *   **"Add New Project"** -> **"Import from GitHub"**를 선택합니다.
    *   방금 올린 리포지토리를 선택하고 **[Deploy]** 버튼을 누릅니다.

3.  **도메인 연결 (선택)**:
    *   `safesettlement.com` 같은 나만의 도메인이 있다면 설정 메뉴에서 연결할 수 있습니다.

## ⚠️ 배포 전 체크리스트

1.  **메인 페이지 이름**: 파일명이 반드시 `index.html` 이어야 합니다.
2.  **링크 경로**: 모든 파일 내의 링크(css, js, 이미지 등)가 상대 경로(`./style.css` 등)로 되어 있는지 확인하세요. 현재 코드는 잘 되어 있습니다.
3.  **모바일 테스트**: 배포 후 반드시 휴대폰으로 접속하여 레이아웃이 깨지지 않는지 확인하세요.
